## 🔭 What I'm Building

I design systems that don't need babysitting. Networks that heal themselves. Code agents that ship code. Businesses that run themselves while I'm asleep.

- 🤖 **Multi-agent AI pipelines** — Claude + Codex + local Ollama, orchestrated via an event-driven bus. My fleet ships production code, reviews PRs, and escalates to humans only when confidence is low.
- 🌐 **Network automation at scale** — Cisco NSO service packages, YANG-modeled configs, ZTP platforms that take provisioning from days → minutes.
- 📡 **Telemetry-driven change intelligence** — signals in, decisions out. Self-healing infrastructure that knows *why* things broke, not just *that* things broke.
- 💼 **Productized automation for businesses** — every system I build is either a client deliverable or a service I sell.

## 🎯 What Makes This Different

- **Enterprise engineering at small-business velocity.** Carrier-grade practices, 2-day delivery cycles.
- **Zero-cloud-dependency architectures.** Runs on a ThinkPad + Mac Mini + one prod box. No AWS bill.
- **Agents that ship code, not just suggest it.** My PR pipeline reviews, tests, merges, and deploys autonomously. Humans triage only the hard calls.
- **Everything is monetizable.** If I built it, a client can buy it — infrastructure, automation, websites, support contracts.
